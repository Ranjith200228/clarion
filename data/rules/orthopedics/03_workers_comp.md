# Workers' Compensation patients

Workers' Comp cases require additional handling.

## At booking

For any patient mentioning workers' comp, on-the-job injury, or an employer-
related injury, the agent must collect:

- Employer name
- Workers' Comp carrier
- Claim number (may be pending — that is OK to note)
- Date of injury
- Whether the patient has an assigned case manager

## Eligibility

Workers' Comp eligibility usually shows up as status "pending" in the
eligibility record while the claim is being adjudicated. The agent should
proceed with booking but always create a task for the front desk to follow
up on claim status before the visit.

## Which providers see workers' comp cases

All providers see workers' comp cases. There is no provider-level
restriction.
